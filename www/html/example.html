Recomendations

Simulator generates node bandwidth between 1 and 2500 KB/s (adjust yours values acordingly)

If you simulate pats selection it si not recomendet to generate too many paths reccomendet +-100 ( becouse final graph is interactive its size would be up to 30MB)

Small graph in path simulation use max 5 of each eaiteh graph will be too big. Its purouse is demonstration. use large...

                
[general]
; Type of simulation: path/hidden_service/attack/exit_attack/multiple_sim
simulation_type = multiple_sim
; removes duplicit paths: True/False
remove_duplicate_paths = False
; All nodes will have same bandwidth: True/False
same_bandwidth = False
; Optional - bandwidth value for guard nodes (KB/s)
guard_bandwidth_value = 500
; Optional - bandwidth value for exit nodes (KB/s)
middle_bandwidth_value = 500
; Optional - bandwidth value for exit nodes (KB/s)
exit_bandwidth_value = 500
; generate graph: True/False (in web aplication always true)
generate_graph = True
; creates HTML file with animated graph: True/False (in web aplication always true)
create_html = True
;Optional - tor path simulator folder: /home/torps (Only if TorPS is not in native location - in web aplication nothing)
path =
